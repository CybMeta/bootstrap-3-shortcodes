Just The Shortcodes, Please

Plenty of great WordPress plugins focus on providing the Bootstrap library. Bootstrap Shortcodes for WordPress assumes you already know how to include Bootstrap in your theme or site and focuses on giving you a great set of shortcodes to use it with.

This plugin creates a simple, out of the way button just above the WordPress TinyMCE editor (next to the "Add Media" button) which pops up the plugin's documentation and examples for reference. There are no additional TinyMCE buttons to clutter up your screen, just great, easy to use shortcodes!

Plugin Documentation

Bootstrap Shortcodes for WordPress is focused on providing great, powerful, easy to use shortcodes, and to documenting those shortcodes thoroughly. The plugin's documentation and example shortcodes are always available in a handy popup window from any WordPress TinyMCE editor.

Requirements

Bootstrap Shortcodes for WordPress is designed to work with Boostrap 3 and WordPress 3.8 and requires PHP 5.3+.

Installation

  1. Download and unzip the plugin from the link at the top of this page
  2. Upload the the bootstrap-3-shortcodes folder to your wp-content/plugins directory.
  3. Activate the plugin in WordPress!
